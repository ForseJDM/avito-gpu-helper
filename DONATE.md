# 💝 Поддержать проект

Avito GPU Helper — полностью бесплатное open-source расширение без рекламы и трекеров.

Если расширение сэкономило вам деньги на покупке видеокарты, или просто нравится — поддержите разработку!

## 🎯 Куда идут деньги

- ** residential proxy для парсера** (Bright Data / Oxylabs) — поднимет покрытие remote prices с 30% до 80%+
- **Домен и хостинг** для лендинга и будущего API
- **Apple Developer Program** (для будущего Safari-порта)
- **Firefox AMO подпись** (для будущего Firefox-порта)
- Время разработчика на новые фичи
